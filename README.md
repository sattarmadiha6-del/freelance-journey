# My Freelancing Journey

A personal page to track achievements, log consistency, and stay motivated.

## Files
- `index.html` — the full app (all HTML, CSS, JS in one file)
- `server.js` — simple Node.js server
- `package.json` — project config

## Deploy on Railway

1. Go to https://railway.app and sign up (free)
2. Click **New Project** → **Deploy from GitHub repo**
3. Push these 3 files to a GitHub repo first, then connect it
4. Railway will auto-detect Node.js and run `npm start`
5. Your site will be live at a `.railway.app` URL

## Run locally (test before deploying)

```bash
node server.js
```
Then open http://localhost:3000 in your browser.

## How to use the app

1. Click **Edit** → add your name, tagline, and WHY you're doing this
2. Tick off days in **This week** every time you study or practice
3. Click **Add Win** whenever you complete something — no win is too small
4. Bookmark the live URL and visit it when you feel like giving up
